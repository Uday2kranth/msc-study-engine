UNIT-I: Web Data Mining & Foundations

Foundations: Introduction to WWW, Web Mining, and Data Mining.

Sequence Matrices: Association Rule Mining - Apriori Algorithm, Frequent Itemset & Rule Generation, Multiple Minimum Supports, Class Association Rules. Sequential Pattern Mining - GSP, PrefixSpan, Rule Generation from Patterns

UNIT- II: Machine Learning for Web Mining

Supervised Layers: Supervised Learming – Decision Trees, Rule Induction, Classification based on Associations, Naïve Bayes & Text Classification.

Unsupervised Sorting: Unsupervised Learming - K-means Clustering, Hierarchical Clustering (Single Link, Complete Link, Average Link), Strengths & Weakmnesses

UNIT-I: Information Retrieval, Link Analysis & Web Crawling

Retrieval Systems: Information Retrieval - Boolean Model, Vector Space Model, Statistical Language Model, Relevance Feedback, Evaluation Measures. Text & Web Page Preprocessing – Stopword Removal, Stemming, Duplicate Detection, Inverted Index & Compression, Latent Semantic Indexing. Web Search & Issues -Web Search, Meta Search, Web Spamming.

Graph Analysis: Link Analysis – PageRank, HITS, Community Discovery.

Crawler Automation: Web Crawling - Crawler Algorithms (BFS, Focused, Topical), Implementation Issues, Ethics. Sentiment Classification – Sentiment Phrases, Text Classification Methods.
