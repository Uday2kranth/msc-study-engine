UNIT -I: Overview of Security & Classical Encryption

Security Architecture & Frameworks: OSI Security Architccture, Security Attacks, Security Services, Security Mechanisms, a Model for Network Security.

Symmetric Ciphers: Classical Encryption Techniques: Symmetric Cipher Model, Substitution Techniques, Transposition Techniques, Rotor Machines, Steganography.

Block Cipher Operations: Block Ciphers:Structure and Data Encryption Standard (DES), Strength of DES. Block Cipher Operation: Double and Triple DES, Electronic Code Book, Cipher Block Chaining Mode, Cipher Feedback Mode, Output Feedback Mode,Counter Mode.

UNIT –II: Encryption Standard (AES) & Public-Key Cryptography

Advanced Symmetric Ciphers: Encryption Standard (AES): Origins, Structure, Round Functions, AES Key Expansion.

Stream Ciphers & Keys: Pseudorandom Number Generation and Stream Ciphers: Principles, Block Cipher based PRNG, RC4.

Asymmetric Systems: Public-Key Cryptography: Principles of Public-Key Cryptosystems, RSA Algorithm. Key Management and Distribution:Symmetric and Asymmetric Key Distribution, Public Key Distribution, X.509 Certilicates, Diffie-Hellman Key Exchange.

UNIT -Ⅱ: Cryptographic Hash Functions, Digital Signatures & System Security

Integrity & Authentication: Cryptographic Hash Functions:Applications, SHA & MDS Algorithm. Message Authentication Codes (MAC): requirements, HMAC, CMAC. Digital Signatures: Concepts, NIST Digital Signatures Algorithm.

Network Layers: Transport-Level Security: SSL, TLS, HTTPS, SSH. E-Mail Security: Pretty Good Privacy, SMIME. IP Security:Overview, Architecture, Encapsulating Security Payload, Intemet Key Exchange.

Defensive Systems: System Security: Intruders, Intrusion Detection, Password Management, Virus and Countermeasures, Firewall Design Principles and Types.

Structural Link Intrusion Anomaly Modeling: The logic of monitoring incoming packets for structural anomalies in an Intrusion Detection System shares a behavioral framework with parsing web attack vectors and spam patterns.
